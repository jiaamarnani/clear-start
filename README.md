# ClearStart 

A personalized skincare web application designed to help teens build simple, effective and accesibile acne treatment routines through an interactive quiz experience. Designed to educate and normalize acne, with love!

## About

ClearStart provides customized skincare recommendations based on user-specific factors including skin type, concerns, and lifestyle. Currently working to implement API based reccomendation system, ML education system and account based tracking. 

## Features

- Interactive Quiz
    - Dynamic questionnaire to assess skin needs
- Personalized Results 
    - Custom acne routine recommendations
- Responsive Design 
    - Optimized for all device sizes

## Tech Stack

- **Next.js 15** 
- **TypeScript** 
- **Tailwind CSS** 
- **React** 

## How to Run

Install dependencies:
```bash
npm install
```

Run the development server:
```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the app.

## Project Structure
```
/app
  /quiz       # Quiz flow pages
  /results    # Results and recommendations
  page.tsx    # Landing page
```
